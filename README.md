# DevSecOps - Aplicación Móvil Gym

Proyecto de implementación **DevSecOps** para una aplicación Android de gestión de gimnasio.

## 🎯 Objetivo
Implementar prácticas DevSecOps completas: CI/CD, Seguridad automatizada, IaC y Monitoreo.

## 🛠 Tecnologías
- **Frontend**: Android (Kotlin + Jetpack Compose)
- **CI/CD**: GitHub Actions
- **Seguridad**: Trivy, Dependency Check, CodeQL
- **IaC**: Terraform
- **Build**: Gradle

## 📁 Estructura del Proyecto
- `/app` → Código fuente de la aplicación
- `/.github/workflows` → Pipelines CI/CD
- `/iac` → Infraestructura como Código (Terraform)
- `/app/src/test` → Pruebas unitarias

## 🚀 Pipeline DevSecOps
El pipeline incluye:
1. Análisis de seguridad (SAST + SCA)
2. Ejecución de pruebas unitarias
3. Build de APK
4. Despliegue de infraestructura
